＜はじめに＞
　初めまして。三柱(さんちゅう)と申します。
此度は本モデルをダウンロードしていただきありがとうございます。
本モデルは、「ウマ娘 プリティーダービー」に登場する「トレセン学園」を原作としモデルアップした二次創作(ファンアート)モデルになります。
モデリングはBlender(ver2.79)、セットアップはPMXエディタ(ver0.2.3.2)で、そして動作確認はMikuMikuDance ver9.31(64bitOS Version)(以下、MMD)で行っています。
　目次以降のテキストをよく読み、楽しいMMDライフをご満喫ください。
