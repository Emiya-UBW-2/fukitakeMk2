なんちゃって競馬場セット by Led/折鶴P

○このモデルについて
競馬場のコース・ゲート・ハロン棒・ゴール板および蹄鉄のセットです。
このモデルに添付しているテクスチャはLed/折鶴Pが撮影した写真から作成されています。
（土のテクスチャは動画sm24302238で使用したテクスチャとは異なります）

○無保証
このモデルは無保証です。このモデルの使用によって発生するいかなる損害も保障されません。


○利用許諾について
モデルの作者表示を誤らない限り、どなたにもご利用いただけます。


○改造・再配布
商用/非商用を問わず改造・再配布に制限は設けません。
mqoファイルを添付していますので、どんどん改造してください。
また、競馬場コースは添付の.cxファイルで生成しました。
