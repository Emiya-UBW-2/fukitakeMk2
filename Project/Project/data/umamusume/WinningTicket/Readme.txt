【追加モーフの説明】

まずAL系モーフはMMEのオートルミナス（発行エフェクト）に対応しており光ります。
次に『涙１』『涙２』ですが、初期位置は瞬きの位置に設定しております。通常時の目が開いている時は、それぞれの涙位置モーフでずらして下さい。
『涙１うるうる』はモーフで動かしながらやると、涙がうるうるしている様に見えます。（固定だとうるうるしません。）
また、『涙２』は涙を流す様に出来ており、縦位置と横位置でうまく操作すれば涙を流している様に出来ます。

胸親ボーンですが、これは結構要望の声が多く追加致しました。
まず、初期設定で今まで通り胸が揺れない状態です。
揺らすには『乳揺れ』モーフを入れて下さい。
そうすると揺れるようになります。（あくまでもアプリと同じ位ですね。）
『垂れ防止』はそのままの意味です。

手の『拡散ボーン』はパーにする事が出来ます。
逆に『握りボーン』はグーにする事が出来ます。物を掴む時にとかにも便利です。
